# Exercises 1.3

## Solution 1

```scheme
(define (fn x y z)
  (cond ((and (> x y) (> y z)) (+ x y))
        ((and (> x z) (> z y)) (+ x z))
        ((and (> y x) (> x z)) (+ y x))
        ((and (> y z) (> z x)) (+ y z))
        ((and (> z x) (> x y)) (+ z x))
        ((and (> z y) (> y x)) (+ z y))))
```

## Solution 2

```scheme
(define (fn x y z)
  (cond ((or (and (> x y) (> y z)) (and (> y x) (> x z))) (+ x y))
        ((or (and (> x z) (> z y)) (and (> z x) (> x y))) (+ x z))
        ((or (and (> y z) (> z x)) (and (> z y) (> y x))) (+ y z))))
```

## Solution 3

```scheme
(define (fn x y z)
  (cond ((and (> x y) (> z y)) (+ x z))
        ((and (> x z) (> y z)) (+ x y))
        (else (+ y z))))
```

## Solution 4

```scheme
(define (min x y) (if (< x y) x y))
(define (fn x y z) (- (+ x y z) (min (min x y) z)))

; block structure
(define (fn x y z)
  (define (min x y) (if (< x y) x y))
  (- (+ x y z) (min (min x y) z)))
```
